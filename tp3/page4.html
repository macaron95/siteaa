Développeur Lua

Lua est un langage de script développé au Brésil au début des années 1990. Le code source d'un programme en Lua est traduit et exécuté sous forme de code byte par un interpréteur Lua. L'interpréteur lui-même est codé en C, ce qui permet aux programmes en Lua de disposer d'une performance élevée lors de l'exécution.
